# OPCUA-Azubi-Projekt
Kommandozeilenprogramm zur Verbindung mit einem OPC UA Server. Es ermöglicht das Auslesen von beliebigen Variablen aus einem OPC UA Simulator oder einem echten Server.  Das Ziel des Projekts ist es, die Grundlagen der industriellen Kommunikation mit OPC UA umzusetzen und ein leichtgewichtiges Tool für Test und Lernzwecke zu entwickeln
